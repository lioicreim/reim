# 프리뷰 배경 이미지 가이드

## 이미지 위치
프리뷰 배경 이미지는 `public` 폴더에 저장합니다.

**경로**: `public/preview-ground.jpg`

## 이미지 파일명
- 기본 파일명: `preview-ground.jpg`
- 관리자모드에서 변경 가능하도록 나중에 설정 파일로 관리 예정

## 이미지 확장자
다음 확장자를 지원합니다:
- `.jpg` / `.jpeg` (권장)
- `.png`
- `.webp` (최적화된 형식, 권장)

## 이미지 사이즈 권장사항
- **최소 해상도**: 1920x1080 (Full HD)
- **권장 해상도**: 2560x1440 (2K) 또는 3840x2160 (4K)
- **비율**: 16:9 또는 21:9 (와이드)
- **파일 크기**: 500KB ~ 2MB (최적화 권장)

## 이미지 스타일
- 게임 바닥/지면 텍스처
- 어두운 톤 (밝은 이미지는 아이템 라벨이 잘 안 보일 수 있음)
- 반복 패턴이 적은 이미지 (배경이 자연스럽게 보이도록)

## 사용 예시
```jsx
<ItemPreviewBox
  itemName="Gold"
  styles={rule.styles}
  backgroundImage="/preview-ground.jpg"
/>
```

## 관리자모드에서 변경하기 (향후 구현)
1. 관리자모드 페이지에서 이미지 업로드
2. 업로드된 이미지는 `public/preview-ground.jpg`로 저장
3. 또는 설정 파일(`data/settings.json`)에 경로 저장
4. 컴포넌트에서 설정 파일의 경로를 읽어서 사용
